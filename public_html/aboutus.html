Experience Developing This Website

Working on this website allowed me to strengthen my skills in web development by:

  • Structuring content with HTML to ensure clarity and usability.
  • Styling and enhancing the design with CSS, including responsive layouts for an optimized user experience on both desktop and mobile devices.
  • Creating static bar charts to visualize data related to Tampa Bay's attractions, without relying on external libraries like Chart.js.

I encountered challenges such as debugging navigation links, ensuring cross-page consistency, and organizing data effectively. Overcoming these challenges deepened my understanding of web development and design principles.

Technologies Used

This project utilized:

  • HTML: For creating and organizing the content across all web pages.
  • CSS: For styling and ensuring visual consistency, including responsive design.
  • Bootstrap: To improve layout and styling for better usability and navigation.
  • Static bar charts for data visualization, designed entirely with CSS, to present insights into Tampa Bay's tourism trends and economic data.
